Postman Collection Importer converts Postman collections and environments into Repeater tabs and Sitemap entries. The extension supports variable resolution, authentication methods, and multiple request body types to facilitate API security testing workflows.

Features

  • Import Postman Collection v2.0 and v2.1 with environment file support and automatic variable resolution
  • Send requests to Repeater for manual testing, Sitemap for live requests with response data, or both simultaneously
  • Preview and selectively import specific requests from collections before sending them
  • Handle Bearer, Basic, API Key, and OAuth2 authentication with support for JSON, form-data, and x-www-form-urlencoded body types
  • Detect and extract GraphQL operations automatically from collection requests
  • Retry failed Sitemap requests with one click and configure rate limiting between requests (0-5000ms)

Usage

  1. Navigate to the Postman Importer tab and click Browse to select your Postman collection .json file
  2. Optionally load a Postman environment file to resolve collection variables
  3. Select the destination: Repeater (manual testing with no HTTP requests), Sitemap (live requests with real responses), or Both
  4. If unresolved variables are detected, choose to upload an additional environment file, manually enter values with smart suggestions, ignore variables, or skip problematic requests
  5. Configure rate limiting delay if using Sitemap mode, then click Import Collection to process the requests
